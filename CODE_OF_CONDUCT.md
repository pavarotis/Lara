Code of Conduct
1. Our Pledge

Για να κάνουμε το Lara ένα ασφαλές και φιλικό περιβάλλον για όλους τους συνεισφέροντες, δεσμευόμαστε ότι θα συμπεριφερόμαστε με σεβασμό, ευγένεια και επαγγελματισμό.





2. Our Standards

Συμπεριφορά που πρέπει να τηρείται περιλαμβάνει:

Ευγένεια και σεβασμός προς όλους τους συμμετέχοντες

Αποφυγή προσβλητικής, απειλητικής ή προσβλητικής γλώσσας

Υποστήριξη διαφορετικών απόψεων με εποικοδομητικό τρόπο

Σεβασμός στην ιδιωτικότητα και στα προσωπικά δεδομένα άλλων





3. Unacceptable Behavior

Δεν θα γίνονται ανεκτά:

Παρενόχληση, bullying ή προσωπικές επιθέσεις

Ρατσιστική, σεξιστική ή μισαλλόδοξη συμπεριφορά

Ανεπιθύμητα μηνύματα, spam ή διαφημίσεις

Δημοσιοποίηση προσωπικών δεδομένων άλλων χωρίς άδεια

Καταχρηστική συμπεριφορά ή sabotage στο project





4. Reporting Issues

Εάν αντιμετωπίσετε οποιοδήποτε πρόβλημα:

Απευθυνθείτε στους maintainers μέσω email ή GitHub issues

Όλες οι αναφορές θα αντιμετωπίζονται εμπιστευτικά

Οι συνεισφέροντες που παραβιάζουν τον κώδικα θα αντιμετωπίζονται ανάλογα (warning → temporary ban → permanent ban)

Email Maintainers: maintainers@example.com





5. Enforcement

Οι maintainers έχουν το δικαίωμα να εφαρμόζουν τον κώδικα σε κάθε κατάσταση, συμπεριλαμβανομένης της αφαίρεσης contributions ή περιορισμού πρόσβασης για να διασφαλίσουν ένα ασφαλές περιβάλλον.
