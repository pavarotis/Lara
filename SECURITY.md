Security Policy
🛡️ Reporting a Vulnerability

Η ασφάλεια του Lara είναι σημαντική. Αν βρείτε οποιαδήποτε ευπάθεια:

1. Στείλτε email στο maintainers@example.com
 με θέμα Lara Security Vulnerability.


2. Περιγράψτε αναλυτικά:

>Τι είδους ευπάθεια βρήκατε

>Βήματα αναπαραγωγής

>Περιβάλλον / έκδοση Laravel / PHP / DB


3. Μην δημοσιοποιείτε την ευπάθεια δημόσια μέχρι να αντιμετωπιστεί.





📝 Supported Versions

Διασφαλίζουμε ότι οι εκδόσεις που υποστηρίζουμε είναι ασφαλείς:

Version	  Status
Main	    Supported
Older	    Security fixes may be limited




🛠️ Security Updates

Όλες οι ευπάθειες αντιμετωπίζονται μέσω patches σε νέα releases

Οι contributors ενημερώνονται για fixes μέσω GitHub releases ή email

Για κρίσιμες ευπάθειες, μπορεί να γίνει emergency release





🔐 Best Practices

Μην ανεβάζετε ευαίσθητα credentials στο repo

Χρησιμοποιείτε .env για όλα τα μυστικά

Ακολουθείτε τις οδηγίες Laravel για secure development: https://laravel.com/docs/security
